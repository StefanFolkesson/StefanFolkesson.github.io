Java Arrayer

Enkla datatyper

Datatyper som inte skrivs med stor bokstav kallas enkla datatyper. int,cher,float, etc.

när du dekarerar en enkal datatyp allokerar du minne till den direkt och variabelnamnet "pekar" på den allokeringen.

Bild1

        
            int tal = 10;
            // tal innehåller 10

Arrayer

En array är en hel hoper med enkla datatyper och namnet på arrayen kan således inte peka på alla variabler på en gång. Så istället så "pekar" variabelnamnet på en array på adressen i minnet där du allokerat utrymmet för arrayen.

Om du inte allokerar något utrymme till arrayen på en gång kommer variabeln peka på null.

        
            int [] arr;
            // arr innehåller null
            int [] brr = new int[10];
            // brr innehåller början på adressen till utrymmet som är allokerat med new int[10]

När du deklarerat en array till int kommer den nu hoppa frammåt en int i minnet för varje adressering du anger:

        
            brr[0] = 10;
            // brr[0] tar adressen i brr och lägger till 0*längden på int för att läsa av/ändra värdet. 
            brr[1] = 20;
            // brr[1] är således brr+1*landen på int och där srkiver vi in 20.
            // brr flyttas aldrig!!! brr pekar alltid på starten av arrayen
