Villkor i Java

Villkor är en viktig del av programmering, eftersom det gör det möjligt att styra programflödet beroende på olika förhållanden. I Java finns det flera olika sätt att hantera villkor, såsom if-satser, switch-satser och loopar.

If-satser

I Java kan du använda if-satser för att utföra olika handlingar beroende på om ett villkor är sant eller falskt. Här är ett exempel på hur du kan använda en if-sats: 

        
            public class Main {
                public static void main(String[] args) {
                    int number = 10;

                    if (number > 0) {
                        System.out.println("Talet är positivt");
                    } else {
                        System.out.println("Talet är negativt");
                    }
                }
            }

Switch-satser

I Java kan du använda switch-satser för att utföra olika handlingar beroende på värdet av en variabel. Här är ett exempel på hur du kan använda en switch-sats: 

        
            public class Main {
                public static void main(String[] args) {
                    int day = 3;

                    switch (day) {
                        case 1:
                            System.out.println("Måndag");
                            break;
                        case 2:
                            System.out.println("Tisdag");
                            break;
                        case 3:
                            System.out.println("Onsdag");
                            break;
                        case 4:
                            System.out.println("Torsdag");
                            break;
                        case 5:
                            System.out.println("Fredag");
                            break;
                        case 6:
                            System.out.println("Lördag");
                            break;
                        case 7:
                            System.out.println("Söndag");
                            break;
                        default:
                            System.out.println("Ogiltigt dagnummer");
                            break;
                    }
                }
            }

Loopar

I Java kan du använda Loopar för att upprepa en viss kod flera gånger. Där använder du villkor för att styra hur många gånger koden ska upprepas. Här är ett exempel på hur du kan använda en for-loop för att skriva ut alla tal från 1 till 10:

        
            public class Main {
                public static void main(String[] args) {
                    for (int i = 1; i <= 10; i++) {
                        System.out.println(i);
                    }
                }
            }
