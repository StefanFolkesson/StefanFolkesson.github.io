Vad är en variabel?

En variabel är en namngiven behållare för att lagra data i ett program. Den kan innehålla olika typer av data, såsom heltal, flyttal, textsträngar och mycket mer. Genom att använda variabler kan vi tilldela värden till dem och sedan använda dessa värden i vårt program.

Användning av nyckelordet var i en deklarering

I Java kan vi använda nyckelordet "var" för att deklarera variabler med infererad typ. Det innebär att vi inte behöver ange den exakta typen vid deklarationen, utan kompilatorn kommer att inferera typen baserat på värdet som tilldelas variabeln. Här är ett exempel: var myVariable = "Hello World"; I detta exempel kommer variabeln "myVariable" att infereras som en String eftersom vi tilldelar den ett textsträngsvärde.

Primitiva datatyper

  • byte: En 8-bitars heltalstyp.
  • short: En 16-bitars heltalstyp.
  • int: En 32-bitars heltalstyp.
  • long: En 64-bitars heltalstyp.
  • float: En 32-bitars flyttalstyp.
  • double: En 64-bitars flyttalstyp.
  • boolean: En boolesk typ som kan vara antingen true eller false.
  • char: En 16-bitars tecken typ.

Referenstyper

  • String: En klass för att representera textsträngar.
  • Array: En samling av element av samma typ.
  • Class: En klass som representerar en typ.
  • Interface: Ett kontrakt som en klass kan implementera.

Exempel på variabeldeklaration och användning

Primitiva datatyper

byte: En 8-bitars heltalstyp. Exempel: byte myByte = 10;

short: En 16-bitars heltalstyp. Exempel: short myShort = 1000;

int: En 32-bitars heltalstyp. Exempel: int myInt = 100000;

long: En 64-bitars heltalstyp. Exempel: long myLong = 1000000000L;

float: En 32-bitars flyttalstyp. Exempel: float myFloat = 3.14f;

double: En 64-bitars flyttalstyp. Exempel: double myDouble = 3.14159;

boolean: En boolesk typ som kan vara antingen true eller false. Exempel: boolean myBoolean = true;

char: En 16-bitars tecken typ. Exempel: char myChar = 'A';

Referenstyper

String: En klass för att representera textsträngar. Exempel: String myString = new String("Hello World");

Array: En samling av element av samma typ. Exempel: int[] myArray = new int[5];

Class: En klass som representerar en typ. Exempel: MyClass myObject = new MyClass();

Interface: Ett kontrakt som en klass kan implementera. Exempel: MyInterface myInterface = new MyImplementation();

Exempel på variabelomvandling

Implicit omvandling: När en mindre datatyp konverteras till en större datatyp. Exempel: int myInt = 10; double myDouble = myInt;

Explicit omvandling: När en större datatyp konverteras till en mindre datatyp. Exempel: double myDouble = 3.14; int myInt = (int) myDouble;
