Java Swing Grundläggande exempel

Introduktion till Java Swing

Java Swing är ett användargränssnittsbibliotek för Java-programmeringsspråket. Det används för att skapa grafiska användargränssnitt (GUI) för Java-applikationer.

Utgångsklassen är JFrame som är själva fönstret som du använder för att bygga upp din applikation.

Det finns en mängd olika komponenter som du kan använda för att skapa ditt GUI, till exempel knappar, textfält, listor och mycket mer.

Java Swing är en del av Java Foundation Classes (JFC) som är en samling av bibliotek för att skapa avancerade GUI-applikationer.

Du kan även rita egna element i Java Swing.

Exempel på Java Swing

Nedan finns ett enkelt exempel på hur man skapar en knapp med Java Swing: 

        
import javax.swing.JButton;
import javax.swing.JFrame;

public class SwingExample {
    public static void main(String[] args) {
        // Skapa en ny JFrame
        JFrame frame = new JFrame("Java Swing Exempel");

        // Skapa en knapp
        JButton button = new JButton("Klicka här");

        // Lägg till knappen i JFrame
        frame.getContentPane().add(button);

        // Ställ in JFrame-egenskaper
        frame.setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE);
        frame.setSize(300, 200);
        frame.setVisible(true);
    }
}

Att arbeta med ett arv istället

Jag föredrar oftast att arbeta med JFrame som ett arv istället då det blir lite mer intuitivt.

        
import javax.swing.JButton;
import javax.swing.JFrame;
public class SwingExample extends JFrame {
    public SwingExample() {
        super("Java Swing Exempel");
        JButton button = new JButton("Klicka här");
        add(button);
        setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE);
        setSize(300, 200);
        setVisible(true);
    }
    public static void main(String[] args) {
        new SwingExample();
    }
}

Samma exampel men med Jframe som ett arv.

Detta är en smaksak och det är bra att känna till båda sätten att arbeta med Swing.

En utgångsklass

Så här kan du skapa en utgångsklass för ditt Java Swing-program:

          
import javax.swing.JFrame;
public class Main extends JFrame {
    public Main() {
        super("Java Swing Exempel"); // Sätter titeln på fönstret
        setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE); // Ser till att krysset stänger fönstret
        setSize(300, 200); // Sätter storleken på fönstret
        setLocation(100, 100); // Sätter positionen på fönstret
        setRezinable(false); // Gör att fönstret inte kan ändra storlek
        setVisible(true); // Gör fönstret synligt
    }
    public static void main(String[] args) {
        new Main(); // Skapar en instans av Main
    }
}
