Literaler i Java

En literal är en fast värde som används direkt i koden. I Java finns det flera olika typer av literaler, såsom heltalsliteraler, flyttalsliteraler, teckenliteraler, strängliteraler och booleska literaler.

Heltalsliteraler

Heltalsliteraler används för att representera heltal i Java. Här är några exempel på heltalsliteraler:

            
                int number1 = 10;
                int number2 = -5;
                int number3 = 0;

Binära literaler

Binära literaler används för att representera heltal i binärt format i Java. Här är några exempel på binära literaler:

            
                int binary1 = 0b1010; // 10
                int binary2 = 0b1111; // 15
                int binary3 = 0b1001; // 9

Octala literaler

Octala literaler används för att representera heltal i oktalt format i Java. Här är några exempel på oktala literaler:

            
                int octal1 = 010; // 8
                int octal2 = 017; // 15
                int octal3 = 021; // 17

Hexadecimala literaler

Hexadecimala literaler används för att representera heltal i hexadecimalt format i Java. Här är några exempel på hexadecimala literaler:

            
                int hex1 = 0xA; // 10
                int hex2 = 0xF; // 15
                int hex3 = 0x11; // 17

Flyttalsliteraler

Flyttalsliteraler används för att representera flyttal i Java. Här är några exempel på flyttalsliteraler:

            
                double number1 = 3.14;
                double number2 = -1.5;
                double number3 = 0.0;

Teckenliteraler

Teckenliteraler används för att representera tecken i Java. Här är några exempel på teckenliteraler:

            
                char letter1 = 'A';
                char letter2 = 'B';
                char letter3 = 'C';

Strängliteraler

Strängliteraler används för att representera strängar i Java. Här är några exempel på strängliteraler:

            
                String str1 = "Hello, World!";
                String str2 = "Java 101";
                String str3 = "Programming";

Booleska literaler

Booleska literaler används för att representera sanna eller falska värden i Java. Här är några exempel på booleska literaler :

            
                boolean bool1 = true;
                boolean bool2 = false;
