Strängar i Java

En sträng är en sekvens av tecken som används för att representera text i ett program. I Java finns det flera inbyggda metoder för att arbeta med strängar, såsom att jämföra strängar, konkatenera strängar och extrahera delar av en sträng.

Skapa en sträng

För att skapa en sträng i Java kan du använda antingen en strängliteral eller en instans av klassen String. Här är några exempel:

        
            // Skapa en strängliteral
            String str1 = "Hello, World!";

            // Skapa en instans av klassen String
            String str2 = new String("Hello, World!");

Strängmetoder

Java har flera inbyggda metoder för att arbeta med strängar. Här är några exempel på vanliga metoder:

  • length(): Returnerar längden på en sträng.
  • charAt(int index): Returnerar tecknet på en specifik position i strängen.
  • substring(int beginIndex, int endIndex): Returnerar en del av strängen bas
