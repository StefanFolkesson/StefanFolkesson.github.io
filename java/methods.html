Java Methods

Methods with Return Type

A method in Java can return a value of a specific data type. Here's an example:

        
            public int add(int a, int b) {
                return a + b;
            }

Methods with Void Return Type

A method in Java can also have a void return type, which means it doesn't return any value. Here's an example:

        
            public void printMessage(String message) {
                System.out.println(message);
            }

Methods with Parameters

Methods in Java can take input parameters. Here's an example:

        
            public int multiply(int a, int b) {
                return a * b;
            }

Explaining Static

Static methods belong to the class rather than to any specific instance of the class. They can be called without creating an instance of the class.

Explaining Final

A final variable can only be assigned a value once. Once assigned, the value of a final variable cannot be changed. A final method cannot be overridden in a subclass. A final class cannot be subclassed.

Declaring Methods in the Same Class

Methods can be declared inside the same class. Here's an example:

If the method calling the method is static the method also has to be static

        
            public class MyClass {
                public static void main(String[] args) {
                    // Method invocation
                    myMethod();
                }
                static void myMethod() {
                    // Method implementation
                }
            }

Declaring Methods in an External Class

Methods can also be declared in an external class. The external class cant be public. And if the method calling the method is static the method has to be static

Here's an example:

        
            public class MyClass {
                public static void main(String[] args) {
                    // Method invocation
                    ExternalClass.myMethod();
                }
            }
            class ExternalClass {
                static void myMethod() {
                    // Method implementation
                }
            }

Declaring Methods in a New Public Class

You can declare a new public class with its own methods in another file. Here's an example:

        
            FILE: MyClass.java
            public class MyClass {
                public static void main(String[] args) {
                    // Method invocation
                    NewClass.myMethod();
                }
            }

            FILE: NewClass.java
            public class NewClass {
                static void myMethod() {
                    // Method implementation
                }
            }

Declaring final variables in an external class

Final variables can be declared in an external class. Here's an example:

        
            FILE: MyClass.java
            public class MyClass {
                public static void main(String[] args) {
                    // Variable invocation
                    System.out.println(ExternalClass.MY_CONSTANT);
                }
            }

            FILE: ExternalClass.java
            public class ExternalClass {
                public static final int MY_CONSTANT = 10;
            }
