In- och utmatning i Java

In- och utmatning är en viktig del av programmering, eftersom det gör det möjligt för användaren att interagera med programmet och för programmet att kommunicera med användaren. I Java finns det flera olika sätt att läsa in data från användaren och skriva ut data till användaren.

Scanner-klassen för inmatning

I Java kan du använda klassen Scanner för att läsa in data från användaren. Här är ett exempel på hur du kan använda Scanner för att läsa in ett heltal från användaren: 

        
            import java.util.Scanner;

            public class Main {
                public static void main(String[] args) {
                    Scanner scanner = new Scanner(System.in);

                    System.out.print("Skriv in ett heltal: ");
                    int number = scanner.nextInt();

                    System.out.println("Du skrev in: " + number);

                    scanner.close();
                }
            }

Problematiken med jämförelse av strängar

När du jämför strängar i Java är det viktigt att komma ihåg att strängar är objekt och att jämförelseoperatorn == jämför referenser, inte innehållet i strängarna. För att jämföra innehållet i strängar i Java måste du använda metoden equals(). Här är ett exempel på hur du kan jämföra strängar i Java: 

        
            public class Main {
                public static void main(String[] args) {
                    String str1 = "Hello";
                    String str2 = "Hello";

                    if (str1 == str2) {
                        System.out.println("Strängarna är lika");
                    } else {
                        System.out.println("Strängarna är olika");
                    }

                    if (str1.equals(str2)) {
                        System.out.println("Strängarna är lika");
                    } else {
                        System.out.println("Strängarna är olika");
                    }
                }
            }
