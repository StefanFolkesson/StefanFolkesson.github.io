Klassen Math i Java

Klassen Math innehåller flera metoder för att utföra matematiska operationer i Java. Här är några exempel på vanliga metoder:

Absolutvärde

Metoden abs() returnerar absolutvärdet av ett tal. Här är ett exempel:

        
            double x = -10.5;
            double result = Math.abs(x);
            System.out.println("Absolutvärdet av " + x + " är " + result);

Avrundning

Metoden round() avrundar ett tal till närmaste heltal. Här är ett exempel:

        
            double x = 10.5;
            long result = Math.round(x);
            System.out.println("Avrundningen av " + x + " är " + result);

Max och Min

Metoderna max() och min() returnerar det största respektive minsta av två tal. Här är ett exempel:

        
            int x = 10;
            int y = 20;
            int max = Math.max(x, y);
            int min = Math.min(x, y);
            System.out.println("Det största talet är " + max);
            System.out.println("Det minsta talet är " + min);

Exponentiering

Metoden pow() returnerar ett tal upphöjt till en viss exponent. Här är ett exempel:

        
            double x = 2;
            double y = 3;
            double result = Math.pow(x, y);
            System.out.println(x + " upphöjt till " + y + " är " + result);

Kvadratroten

Metoden sqrt() returnerar kvadratroten av ett tal. Här är ett exempel:

        
            double x = 16;
            double result = Math.sqrt(x);
            System.out.println("Kvadratroten av " + x + " är " + result);

Trigonometriska funktioner

Klassen Math innehåller också metoder för att beräkna trigonometriska funktioner, såsom sin(), cos() och tan(). Här är ett exempel:

        
            double x = Math.PI / 2;
            double sin = Math.sin(x);
            double cos = Math.cos(x);
            double tan = Math.tan(x);
            System.out.println("Sin(" + x + ") = " + sin);
            System.out.println("Cos(" + x + ") = " + cos);
            System.out.println("Tan(" + x + ") = " + tan);

Avrundning uppåt och nedåt

Metoderna ceil() och floor() avrundar ett tal uppåt respektive nedåt till närmaste heltal. Här är ett exempel:

        
            double x = 10.5;
            double y = 10.1;
            double z = 10.9;
            double ceil = Math.ceil(x);
            double floor = Math.floor(x);
            System.out.println("Uppåt: " + ceil);
            System.out.println("Nedåt: " + floor);
            ceil = Math.ceil(y);
            floor = Math.floor(y);
            System.out.println("Uppåt: " + ceil);
            System.out.println("Nedåt: " + floor);
            ceil = Math.ceil(z);
            floor = Math.floor(z);
            System.out.println("Uppåt: " + ceil);
            System.out.println("Nedåt: " + floor);

Random

Metoden random() returnerar ett slumpmässigt tal mellan 0 och 1. Här är ett exempel:

        
            double random = Math.random();
            System.out.println("Slumptal: " + random);
            double random2 = Math.random() * 90 + 10;
            System.out.println("Slumptal mellan 10 och 100: " + random2);

Klassen Math innehåller flera andra metoder för att utföra matematiska operationer i Java. För en fullständig lista över metoder, se Java API.
