Loopar i Java

En loop används för att upprepa en sekvens av instruktioner flera gånger. I Java finns det flera olika typer av loopar, såsom for-loopar, while-loopar och do-while-loopar. Här är några exempel på hur du kan använda loopar i Java:

For-loopar

En for-loop används för att upprepa en sekvens av instruktioner ett bestämt antal gånger. Här är ett exempel på hur du kan använda en for-loop:

        
            public class Main {
                public static void main(String[] args) {
                    for (int i = 0; i < 5; i++) {
                        System.out.println("Hello, World!");
                    }
                }
            }

While-loopar

En while-loop används för att upprepa en sekvens av instruktioner så länge ett villkor är sant. Här är ett exempel på hur du kan använda en while-loop:

        
            public class Main {
                public static void main(String[] args) {
                    int i = 0;
                    while (i < 5) {
                        System.out.println("Hello, World!");
                        i++;
                    }
                }
            }

Do-while-loopar

En do-while-loop används för att upprepa en sekvens av instruktioner minst en gång, och sedan så länge ett villkor är sant. Här är ett exempel på hur du kan använda en do-while-loop:

        
            public class Main {
                public static void main(String[] args) {
                    int i = 0;
                    do {
                        System.out.println("Hello, World!");
                        i++;
                    } while (i < 5);
                }
            }

Nästlade loopar

Du kan använda nästlade loopar för att upprepa en sekvens av instruktioner flera gånger, där varje loop innehåller en annan loop. Här är ett exempel på hur du kan använda nästlade loopar i Java:

        
            public class Main {
                public static void main(String[] args) {
                    for (int i = 0; i < 5; i++) {
                        for (int j = 0; j < 5; j++) {
                            System.out.print("*");
                        }
                        System.out.println();
                    }
                }
            }
