Git Lektion - First Lesson

För att starta upp en git lokalt. Via powershell Öppna upp powershell Göra en global konfigurering på namn och epost. se till att anvädnda det namn du har på Github och den epost du kopplat dit.

git config --global user.email "stefan.folkesson@ntig.se" git config --global user.name "StefanFolkesson"

För att kontrollera vilka inställningar du har just nu.

git config --global -l

Skapa mappen du vill ha din repository i

mkdir repon

eller skpa den i explorer se till att du "står" i mappen i powershell

cd sökvägen

när du står i rätt mapp kan du starta upp (initiera) en repository

git init

Skapa en fil att lägga in i repon enklaste är att gå in i utforskaren och skapa en fil. kalle.txt eller liknande. sedan skall du lägga till filen till din repo

git add kalle.txt

eller

git add *

detta väljer alla filer i mappen. du skall nu "spara" ned din första repo det gör man med kommandot commit.

git commit -m "first"

du måste ha ett namn på commiten försök att vara någorlunda beskrivande.

(skapa en repo på Github)

nu har du gjort det du behöver lokalt. nu skall vi skapa en tom repo på github. logga in på github och skapa en ny repo. OBS! den måste vara tom så lägg inte till någon readme eller någon git ignore. kopiera namnet på din repo. exempel (https://github.com/StefanFolkesson/anewrepo.git)

tillbaka till den lokala repon vi skall nu koppla den till github repon.

git remote add origin https://github.com/StefanFolkesson/anewrepo.git

för att kontrollera så allting blev ok kan man skriva

git remote -v

till sist skall vi pusha vår lokala repo till github det gör vi med push# detta kommando är det du använder så fort du vill lägga upp din repo på github.

git push origin master

Branches

Git branches are used to develop features isolated from each other. The master branch is the "default" branch when you create a repository. Use other branches for development and merge them back to the master branch upon completion.

git checkout -b feature_x git checkout master git checkout -d feature_x // tar bort git push origin feature_x //annars finns den inte på hub
