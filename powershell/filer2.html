Powershell Filhantering version 2

Läsa filer

Om man vill läsa vilka filer som ligger i en mapp använder man Get-ChildItem.

Med parametrarna -Path talar man om var man vill leta efter filer(.\ betyder här).

Med parametrarna -Filter talar man om vilka filer man vill se (*.ps1 betyder alla filer som slutar på .ps1).

Exempel:

        
            Get-ChildItem -Path .\ -Filter *.ps1

Skapa filer

När man skall skapa filer i powershell använder man New-Item den har några parametrar

  • -ItemType Directory för att skapa en katalog
  • -Path .\test för att skapa katalogen
  • -Force för att skapa katalogen även om den redan finns
  • -ItemType File för att skapa en fil

Exempel:

   
        
            New-Item -ItemType Directory -Path .\test -Force
            Write-Host "Målkatalogen har skapats."

Flytta filer

För att flytta filer från en mapp till en annan

Exempel:

        
            Get-ChildItem -Path .\ -Filter *.ps1 | ForEach-Object {
                Move-Item -Path $_.FullName -Destination .\test -Verbose
            }

Här ser ni även hur man arbetar med pipelining i powershell. ( | )

Detta innebär att jag för varje pbject som kommer ut från Get-ChildItem kommer att köra Move-Item.

Och $_.FullName är en inbyggd variabel som innehåller sökvägen till filen.
