Powershell Djupare Kunskaper

Loopar

Loopar används för att upprepa en viss kodblock flera gånger. Här är tre exempel på en loopar:

while-loop

        
            $counter =0;
            while($counter -le 10) {
                Write-Host $counter
                $counter++;
            }

for-loop

        
            for ($i = 1; $i -le 5; $i++) {
                Write-Host "Detta är iteration $i"
            }

foreach-loop

        
            $fruits = "Apple", "Banana", "Orange"

            foreach ($fruit in $fruits) {
                Write-Host "I like $fruit"
            }

Villkor

Villkor används för att utföra olika handlingar beroende på om ett visst villkor är sant eller falskt. Här är ett exempel på en if-sats:

        
            $number = 10
            if ($number -gt 5) {
                Write-Host "Talet är större än 5"
            } else {
                Write-Host "Talet är mindre än eller lika med 5"
            }

Funktioner

Funktioner används för att organisera och återanvända kod i PowerShell. Här är ett exempel på hur du definierar och använder en funktion:

        
            function SayHello($name) {
                Write-Host "Hej, $name!"
            }
            
            SayHello "Alice"

Mer Powershell Hantering

Här är mer exempel på Powershell-hanteringskod:

        
            # Skapa en ny mapp
            New-Item -ItemType Directory -Path "C:\Mapp"

            # Skapa en ny fil
            New-Item -ItemType File -Path "C:\Mapp\fil.txt"

            # Kopiera en fil
            Copy-Item -Path "C:\Mapp\fil.txt" -Destination "C:\Mapp\kopia.txt"

            # Flytta en fil
            Move-Item -Path "C:\Mapp\fil.txt" -Destination "C:\Mapp\NyMapp\fil.txt"

            # Ta bort en fil
            Remove-Item -Path "C:\Mapp\fil.txt"

            # Skapa en ny textfil och skriv till den
            Set-Content -Path "C:\Mapp\textfil.txt" -Value "Detta är innehållet i textfilen"
